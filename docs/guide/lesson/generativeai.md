# 🧠 生成式人工智能导论

> 课程篇均为个人看法，不代表官方答案，如有疑问请咨询专业课老师。


恭喜你，已经掌握了基础的文档与环境配置，现在是时候正式踏入生成式AI的奇妙世界了！

**如果说之前的技能是教你如何“使用工具”，那么这一趴的内容，将带你去“理解魔法”。我们将要接触的 ComfyUI 和 Dify，虽然名字听起来有点技术范儿，但它们其实是你通往AI核心世界的两把“钥匙”。**

在这里，你不再仅仅是那个对着AI说“画一只猫”的用户，而是即将成为那个设计规则、搭建流程的创造者。面对这些新概念，感到一丝陌生和好奇是完全正常的，毕竟我们即将探索的是目前最前沿的技术领域。

## 🔧 模块一：ComfyUI —— 拆解AI绘画的“可视化引擎”
学习目标：深入理解Stable Diffusion的工作流与数据逻辑。
你将不再只是一个点击“生成”按钮的用户。通过ComfyUI的节点式界面，你将能像组装电路一样，清晰看到提示词、模型、采样器、解码器每一个环节如何串联与协作。这是你理解复杂AI系统背后数据流向与控制逻辑的绝佳方式。

## 🚀 模块二：Dify —— 组装AI应用的“乐高积木”
学习目标：体验从模型到可用服务的关键一步——应用工程化。
当你理解了模型如何工作，下一步就是让它为更多人服务。Dify让你无需深陷代码，就能通过可视化工作流，将大模型能力与你的数据、工具相结合，快速组装出聊天机器人、智能助手或内容创作应用。这能让你直接聚焦于应用逻辑与用户体验，理解如何让AI真正产生价值。

**不要被看似复杂的界面吓退，所有的大师都是从点亮第一个节点开始的。准备好开启这段探索之旅了吗？让我们一起动手，去拆解并重组这些智能的奇迹吧。**
