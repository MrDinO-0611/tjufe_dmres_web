# Visual Studio Code 安装指南

## 下载
推荐从[官网](https://code.visualstudio.com/)下载...
如果是Windows系统，也可以直接在Microsoft Store中下载。
**不要在电脑品牌的应用商店中下载（如联想应用商店）**

## 配置
1.  安装中文语言包
2.  推荐插件：
    - Python
    - Java Extension Pack
    - Prettier
    - Dracula Theme Official
    - Indent-rainbow
    - Lingma
    - Live Preview
    - Stylelint
