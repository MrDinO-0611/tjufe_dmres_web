# Markdown入门指南（面向大一新生）

欢迎来到 Markdown 与 LaTeX 的世界！本指南专为数管大一新生编写，帮助你快速掌握在学术写作、笔记整理和文档排版中常用的两种轻量级标记语言：**Markdown**（用于日常文档）和 **LaTeX**（用于数学公式与科技排版）。你将在许多课程平台（如 GitHub、GitLab、Obsidian、Typora、Docsify 等）中频繁使用它们。

---

## 一、Markdown 基础语法

Markdown 是一种轻量级的标记语言，用简单符号代替复杂的格式命令，最终可转换为 HTML 或 PDF。以下是最常用语法：

### 1. 标题

使用 `#` 表示标题，`#` 越多，标题层级越低：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

> 💡 建议：一篇文档通常只有一个 `#` 一级标题（即文章标题）。

---

### 2. 段落与换行

- 段落之间需空一行。
- 行尾加两个或以上空格可强制换行（但不推荐，建议用 `<br>` 或空段落更清晰）。

```markdown
这是第一段。

这是第二段。  
这是同一段的第二行（行尾有两个空格）。
```

---

### 3. 强调（粗体、斜体）

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___
~~删除线~~
```

效果：  
*斜体*、**粗体**、***粗斜体***、~~删除线~~

---

### 4. 列表

#### 无序列表

使用 `-`、`+` 或 `*`：

```markdown
- 项目一
- 项目二
  - 子项目（缩进两个空格）
  - 另一个子项目
```

#### 有序列表

使用数字加点（实际渲染时会自动编号）：

```markdown
1. 第一步
2. 第二步
   1. 子步骤
   2. 又一个子步骤
```

---

### 5. 链接与图片

#### 链接

```markdown
[百度](https://www.baidu.com)
<https://www.github.com>  <!-- 自动变成链接 -->
```

#### 图片

```markdown
![替代文字](图片路径 "可选标题")
```

例如：

```markdown
![校徽](/images/logo.png "我校校徽")
```

> ⚠️ 注意：图片路径可以是本地相对路径或网络 URL。

---

### 6. 引用

使用 `>` 表示引用块：

```markdown
> 这是一段引用。
> 可以跨多行。
>
> > 还可以嵌套引用。
```

---

### 7. 代码

#### 行内代码

用反引号包裹：

```markdown
使用 `print()` 函数输出内容。
```

#### 代码块

用三个反引号 ``` 包裹，并可指定语言以高亮：

```markdown
```python
def hello():
    print("Hello, World!")
```


支持的语言包括：`python`、`java`、`c`、`cpp`、`javascript`、`html`、`bash` 等。

---

### 8. 分隔线

使用三个或以上 `-`、`*` 或 `_`（单独一行）：

```markdown
---
```

---

### 9. 表格

使用 `|` 和 `-` 创建表格：

```markdown
| 姓名 | 学号     | 专业       |
| ---- | -------- | ---------- |
| 张三 | 20230001 | 计算机科学 |
| 李四 | 20230002 | 数学       |
```

对齐方式（可选）：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| A      |    B     |      C |
```

---

### 10. 转义字符

若要显示 Markdown 符号本身（如 `*`、`_`、`#`），在其前加反斜杠 `\`：

```markdown
\*这不是斜体\*
```
